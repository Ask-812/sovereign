## The Thesis

For 200 years, creating economic value required: **Idea × Team × Capital × Time.**

AI just collapsed three of those variables. One person + AI now produces what a 15-person team produced in 2020. But the entire economic system — banks, insurance, procurement, compliance — is still built around the **company** as the unit of production.

**Sovereign is the infrastructure layer for the post-corporate era.** We make "one person + AI" a first-class economic entity with:

- ⚡ **60-second entity formation** — optimal legal structure for your jurisdiction, auto-filed
- 🏦 **Instant financial stack** — bank account, multi-currency payments, invoicing
- 📜 **AI-generated contracts** — MSAs, NDAs, SOWs customized in seconds
- 🤖 **6 AI operations agents** — bookkeeper, tax, legal, sales, support, procurement
- 🛡️ **Pooled insurance** — E&O + cyber liability at network-wide rates
- ✅ **Trust Profile** — verified track record that replaces "team size" as credibility
- 🌐 **Network power** — collective vendor pricing, cross-referrals, shared brand trust

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** TailwindCSS v4
- **Fonts:** Geist Sans + Geist Mono
- **Deploy:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
